# Profile React Project


A **React** project for displaying user profile information.  
Built with **Vite**, the project uses **separated components** to display personal details, stats, and different sections of a profile.

---

## Features

- Display user personal information
- Show profile statistics and cards
- Responsive design using CSS (or Tailwind/Bootstrap)
- Component-based structure for easier development
- Ready to deploy on **Netlify**

---

## Prerequisites

- Node.js version 22.x or higher
- npm version 10.x or higher
- Git (for cloning the repository)
